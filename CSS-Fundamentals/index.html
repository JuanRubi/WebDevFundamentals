The Basics

Today we will be learning the basics of cascading style sheet (CSS). This will allow us to style our website and make change the layout.

Color

With css we can change the color of any text and our html

Basic structure:

h2 {

color: royalblue;

}

css selector {

property: value;

}

Classes

Classes allow us to target specific tags. This way we don't change the style of all h1 or all p tags or any other tag.

Basic structure:

Inside any tag add: class="className"

.className {

property: value;

}

ID Selectors

id selectors work very similar to classes. However we should only apply the same id to one tag at most.

Basic structure:

Inside any tag add: id="idName"

#idName {

property: value;

}
