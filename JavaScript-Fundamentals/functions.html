Learning Functions

Functions store JavaScript code that won't execute unless we tell it to.

JavaScript has many built in functions which can be very useful to us.

Useful Functions

  • indexOf("x") - This functions returns the index of x inside a variable.
  • isNaN(x) - Checks to see if x is not a number. Returns boolean.
  • sort() - Sorts an array alphabetically.
